# PIVShell
PIVShell is a Powershell integration of the Pivotal (Download) API
it allows you to Dynamically retrieve content from Pivotal Network

## Getting started

### install PIVPosh
```Powershell
Install-Module PIVPosh -Scope CurrentUser -Force
```
### load the Module

```Powershell
ipmo .\PIVPosh
```
